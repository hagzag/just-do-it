How we can build a cloud native application using the 15-factor application principles ?
Let's start with the 12-factor application principles and then we will see how we can extend it to the 15-factor application principles.

<!-- {{< image src="images/15factor.png" caption="" alt="15 facrtor original icons" height="" width="" position="center" command="fill" option="q100" class="img-fluid" title="image title"  webp="false" >}} -->

The 15 prinicpleas principles are based on the 12-factor application principles.

## 12-factor application principles
1. Codebase
2. Dependencies
3. Config
4. Backing services
5. Build, release, run
6. Processes
7. Port binding
8. Concurrency
9. Disposability
10. Dev/prod parity
11. Logs
12. Admin processes

## The additional principles are:

1. Api 1st
2. Observability
3. Authentication & Authorization
